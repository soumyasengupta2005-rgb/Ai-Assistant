# JARVIS Core AI

A voice-driven AI assistant inspired by JARVIS from Iron Man, built using React, TypeScript, and the Web Speech API.

---

## Live Demo

https://jarvis-core-ai.vercel.app

---

## Features

* Voice Interaction
  Speak to the assistant using your microphone with real-time speech recognition.

* Text-to-Speech Responses
  Dynamic voice replies using browser speech synthesis.

* Multiple Personalities

  * JARVIS – Formal, calm, precise
  * FRIDAY – Efficient and responsive
  * ARIA – Friendly default assistant

* Dynamic Interface
  Animated orb UI with mood-based visual feedback for listening, thinking, and speaking states.

* Task Handling
  Basic task creation and assistant memory.

---

## Tech Stack

* Frontend: React, TypeScript
* Build Tool: Vite
* Voice APIs: Web Speech API
* Styling: Tailwind CSS
* Deployment: Vercel

---

## Compatibility Notes

* Best experienced on Chrome or Edge
* Voice quality depends on the browser and device
* Mobile browsers may have limited voice options

---

## Local Setup

```bash
git clone https://github.com/soumyasengupta2005-rgb/AI-Assistant.git
cd AI-Assistant
npm install
npm run dev
```

---

## Production Build

```bash
npm run build
```
